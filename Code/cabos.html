O que são os cabos submaritimos (intercontinentais)?

Cabos Submarinos são cabos geralmente compostos de fibra óptica que são instalados no fundo do oceano para transmissão de dados gerais de nível intercontinental,sendo o mais utilizado meio de comunicação e compartilhamento de dados entre continentes,é muito importante para o funcionamento da internet mundials. Eles são essenciais para a internet global, transportando a maior parte do tráfego de dados internacionais, como e-mails, páginas da web e chamadas de vídeo.
